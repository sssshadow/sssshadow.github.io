使用Docker编译AOSP

前言

​ 为了日常使用，我的干活电脑还是装了Deepin，一个不错的Linux发行版，我常用的软件QQ微信网易云搜狗都有了。虽然还是会有些小小的BUG，但是总的来说作为一个Linux功能还是挺丰富的。

​ 但是前一阵子编译AOSP的时候，发现缺少了很多库，编译报错，一个个解决又实在是麻烦，后来看到大家都在使用docker编译啦，这就完美的解决了环境配置等杂七杂八的原因，确实是个好方法。

Docker

​ 这里我使用了别人配置好发布的Docker，连接在这，按照教程下载下来，然后把安卓源码挂载到docker里就行了。

​ 教程写的很详细了，我这也没啥好写的，就当记录个连接吧。

​ docker中有好几个java版本，根据要编译的源码版本切换到不同的java版本，然后按照正常的编译流程走就行了。线程多开点快的话半个多小时就能编译完了（#滑稽）

​ 如果内存太小可能会遇到内存不够的报错，把内存上限调高点就行了。
